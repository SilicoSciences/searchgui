SearchGUI

graphical user interface for proteomics identification search engines

SearchGUI is a user-friendly, lightweight and open-source graphical user interface for configuring and running proteomics identification search engines, namely X!Tandem, MS-GF+, MS Amanda, MyriMatch, Comet, Tide, Andromeda and OMSSA.

For more details and help see the SearchGUI home page: http://searchgui.googlecode.com.

To visualize and analyze SearchGUI results we recommend the use of PeptideShaker.

If you find any bugs or have any questions please contact the developers.

Please cite SearchGUI as Vaudel et al.: Proteomics 2011;11(5):996-9.
